# BCIT Campus PWA

**BCIT Campus PWA** is a Progressive Web App (PWA) that allows users to explore different BCIT campuses, view campus maps, and read essential information about facilities. This project demonstrates a modern, installable web app with offline capabilities.

## Features

- **Responsive Design** – Works on both desktop and mobile devices
- **Campus Selector** – Select from Burnaby, Downtown, Marine, Aerospace Technology, and Annacis Island campuses
- **Interactive Maps** – Display the map of the selected campus
- **Offline Support** – Service Worker caches content for offline use
- **Installable** – Can be added to home screen like a native app
- **Clean UI** – Modern, professional interface inspired by BCIT branding
